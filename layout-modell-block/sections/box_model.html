Box Model

Die Breite eines Block-Level Elements errechnet sich aus:
Außenabstand links + Rand links + Innenabstand links + Breite + Innenabstand rechts + Rand rechts + Außenabstand rechts

Die Höhe eines Block-Level Elements errechnet sich aus:
Außenabstand oben + Rand oben + Innenabstand oben + Höhe + Innenabstand unten + Rand unten + Außenabstand unten

Es gilt: Der Außenabstand kann negativ sein. Alle anderen Eigenschaften nehmen nur positive Werte oder das Schlüsselwort auto an.

Die Grafik veranschaulicht in welcher Reihenfolge Außenabstand, Rand und Innenabstand vom Browser gezeichnet werden.
